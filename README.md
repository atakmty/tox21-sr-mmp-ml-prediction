# SR-MMP Toxicity Prediction Using Morgan ECFP Fingerprints and Machine Learning

## Project Overview
Machine learning models for predicting mitochondrial membrane potential (SR-MMP) 
toxicity using the Tox21 dataset. Implements Morgan ECFP fingerprints with 
PCA, KNN, SVM, Decision Trees, Naive Bayes, Neural Networks, and ensemble methods.

## Dataset
- **Tox21 SR-MMP**: Mitochondrial membrane potential assay
- **Format**: SMILES notation
- **Features**: Morgan ECFP fingerprints (2048 bits)

## Models
- KNN (k-Nearest Neighbors)
- SVM (Support Vector Machine)
- Decision Trees
- Naive Bayes
- Neural Networks
- Ensemble Methods (Random Forest, Gradient Boosting)
